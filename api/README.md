# Physical AI & Humanoid Robotics - RAG Chatbot API

A Retrieval-Augmented Generation (RAG) chatbot for the Physical AI & Humanoid Robotics documentation.

## Features
- Answer questions about robotics concepts
- Context-aware responses based on selected text
- Source citations from documentation

## API Endpoints
- `POST /api/chat` - Send a message
- `GET /api/health` - Health check
